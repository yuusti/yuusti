Copilotに作ってもらったアルファベットと数を対応付けするための修行ゲー

## 指示文
```
github pages で動作するアプリケーションのコードを生成してほしいです。
アプリケーション上では、ユーザーは以下のようなゲームを行います。
アルファベットと数がaと1, bと2, ... , zと26というように対応しているとして、
ランダムなアルファベットまたは数を表示されるので、
アルファベットが表示されたら対応する数を入力する。
数が表示されたら対応するアルファベットを入力する。
不正解だった場合は正解の値を教える。

また、追加の要件は以下の通りです。
エンターで入力を送信できるようにする。
結果の履歴を以下のようなテーブルで表示する。
履歴は新しいものが上にくるようにする。
正解していた場合は行の背景を薄い緑に、不正解だった場合は行の背景を薄い赤にする。

||問題|入力|正解||
|a|2|1|
|d|4|4|
...

前回のコードに、解答にかかった時間を履歴に含めるようにしてください。
```
